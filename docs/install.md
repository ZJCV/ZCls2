    Python 3.9

    torchvison 0.9.2 or higher

## Install ZCls2

### Source install

* Install `pytorch/torchvision/apex` following the official instructions.

* Clone the `ZCls2` repository.

```
git clone https://github.com/ZJCV/ZCls2.git
cd ZCls2
```

* Install `ZCls2`.

```
python3 setup.py install
```

### Pip install

```python
pip3 install zcls2
```
